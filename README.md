# Морской бой - SeaBattle #
Небольшой проект онлайн морского боя. 

[Онлайн версия на heroku](https://seabattlevan.herokuapp.com/) 
## На чём это работает? ##
1. vue.js
2. socket.io
3. node.js - express

## Использование ##
Для компиляции библиотек

    npm run build
Для запуска сервера

    node server

По умолчанию сервер запустится по адресу **http://localhost:3000**
